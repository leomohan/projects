## Local Development

```bash
# Clone the repository
git clone https://github.com/leomohan/projects.git
cd projects

# Open in your browser
open index.html
# or
python -m http.server 8000
```

Then visit `http://localhost:8000`

## Adding a New Project

1. Create a new subdirectory with your project
2. Add project files and documentation
3. Update index.html to include the new project link
